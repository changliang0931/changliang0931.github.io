🔌 DTU安装接线指南 外置版
1 接线端子定义（外置版）

📌 外置版DTU接线端子（从左到右）：

  • 485- → RS485 负极 (B-)
  • 485+ → RS485 正极 (A+)
  • COM → 公共地
  • 24V → 电源正极 (DC 24V)
✅ 供电说明： 支持 DC 9~36V 宽压输入，推荐使用 24V 工业电源。
2 DTU ↔ 变频器 接线对照表

🔗 标准接线方式：

线色 DTU端 变频器端
🔵 蓝色 / 白色 485- 485- / 485B / B- / S-
🟢 绿色 / 黄色 485+ 485+ / 485A / A+ / S+
⚫ 黑色 COM COM / GND
🔴 红色 24V 24V / V+
💡 注意： 变频器端标示可能为 485+ → 485A / A / A+ / S+，
485- → 485B / B / B- / S-，功能一致，请按实际端子名称对应。
⚡ 通讯干扰处理 故障排查

🔴 干扰典型表现

  • 表现一： 设备运行时，小程序读取数据显示 65537 或报 65537 故障
  • 表现二： 小程序能正常开启设备，但关闭设备时偶尔不成功
3 干扰解决方案

🔧 解决方法：

  • 使用导线连接 变频器端子上的 GND 与 DTU 的 COM 端口
  • 确保屏蔽层在单端接地（推荐在DTU侧接地）
  • 检查通讯线是否使用双绞屏蔽线
⚠️ 操作前请断开设备电源，确保安全！
📶 4G信号检测指南 信号优化
4 信号强度判断与优化

① 查看指示灯： NET灯常亮 = 4G在线；快速闪烁 = 搜网中；熄灭 = 未注册

② 后台读取信号值： 登录云平台 → 设备详情 → 查看 RSRP / RSSI

  • RSRP > 26 → 信号优 ✅
  • RSRP 20 ~ 25 → 信号中等
  • RSRP < 19 → 需优化天线

③ 天线微调： 缓慢转动天线方向或延长引出机柜，实时观察信号值变化

📡 偏远地区信号弱时，可选用高增益玻璃钢天线或4G信号放大器

🔗 一物一码 快速配置

每台DTU外壳均有专属二维码，扫码立即获取设备参数模板及远程启停入口，售后无需线下奔波，云端一键完成配置下发。

常见问题❓

  • Q： 485通讯不通？ → 检查A/B线序是否接反，同一总线设备需共地。
  • Q： 遥控器无法控制设备？ → 检查遥控器地址码与DTU配置是否一致。
  • Q： 出现65537故障？ → 按第3步方法连接GND与COM，消除干扰。
  • Q： 偏远地区信号弱？ → 参考第4步信号检测指南，优化天线位置或加装放大器。
