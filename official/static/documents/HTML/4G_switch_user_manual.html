产品概述

MKK-CB-T4S80 4G智能三相电源保护器是一款集电力保护、电机保护与远程通信于一体的智能设备。通过4G Cat.1频段实现数据传输，支持云平台远程监控和小程序远程操控，适用于工业电机、配电系统、水泵机组等设备，提供全面的电气故障保护与智能管理功能。

核心功能

  • 4G Cat.1无线通信，内置流量卡
  • 兼容三相三线、三相四线
  • 小程序远程监控、远程操控
  • 支持多种定时控制模式
  • 过流保护、漏电保护
  • 过欠压保护、过温保护
  • 专项电机保护

应用领域

  • 水泵、风机电机保护
  • 智能灌溉与农业物联网
  • 智能电网与能源管理
  • 高谐波环境用电保护
  • 其他电机保护场景

安装与接线指南

接口定义

设备采用4P接口，定义如下：

引脚 定义 描述
N 零线 零线，三相三线时可不接零线
A L1火线
B L2火线
C L3火线

接线步骤

确保设备处于断电状态；
将三相电源线接至进线端的N、A、B、C端口；
将负载电源线接至出线端的N、A、B、C端口；
拧上天线并将天线拉到铁箱外，置于易接收信号处
仔细检查所有接线是否牢固，确保无短路风险；
接通电源，观察设备指示灯状态。
安全警告：
  • 接线前请确保设备完全断电，防止触电或设备损坏
  • 确保电源电压在额定范围内(AC 380V±10%（50/60Hz）)。

指示灯说明

指示灯 颜色 状态 含义
工作灯 绿灯 常亮 开关处于断开状态
工作灯 红灯 常亮 开关处于闭合状态
网络灯 蓝灯 常亮 4G通讯工作正常
网络灯 蓝灯 闪烁 4G网络搜索中

按键说明

短按：控制开关通断
长按10s以上：重启保护器

设备保护功能配置

通用保护功能

功能 描述
过流保护 电流超过设定阈值时（0.1~80A可调），自动切断电路，响应时间≤0.1s。
漏电保护 检测漏电流≥50mA（可调）时触发保护切断电路，响应时间≤0.1s。
过压/欠压保护 相电压超限（过压：>450V（可调），欠压：<330V（可调））时动作，阈值可调。
过热保护 内置温度传感器，设备温度＞65℃（可调）时自动断电。
重合闸功能 故障发生后自动合闸恢复供电（延时300s可调）

电机保护设置

功能 描述
电机过载保护 基于反时限特性，过载电流倍数越高，动作时间越短（120%（可调）额定电流60s（可调））。
电机空载保护 空载电流持续低于额定值50%（可调）时，判定为空载并切断电源。
电机堵转保护 堵转电流≥400%（可调）额定电流时，5s（可调）内切断电源。
缺相保护 任意一相断相时，1s内动作。
电流不平衡保护 三相电流中最小电流低于最大电流的80%（可调）时触发保护，防止绕组烧毁。

维护与故障排除

日常维护

  • 定期检查设备连接线是否牢固
  • 保持设备表面清洁，防止灰尘积聚
  • 避免设备处于极端温度或湿度环境中
  • 定期检查天线连接状态

常见问题处理

问题现象 可能原因 解决方法
工作灯不亮 电源未接通或电源故障 检查电源连接和电压
网络灯一直闪烁 SIM卡过期或网络信号弱 小程序查看流量卡到期时间或调整天线位置

2跳闸故障处理

故障代码 故障类型 处理建议
E001 电流过大 瞬间电流超过跳闸电流，请检查负载是否过大或存在短路
E002 漏电故障 请检查负载是否存在漏电情况
E003 欠压故障 请检查进线电压是否过低，如属于接线过长导致的正常压降，也可调整欠压点设置
E004 过压故障 请检查进线电压是否过高
E005 输入缺相 请检查进线的三相电压是否存在缺相
E006 模块过热 请是否存在电线虚接导致的发热
E007 过载保护 请检查电机电流参数是否正确设置，若参数设置正确，再检查负载是否存在超负荷工作情况
E008 空载保护 请检查电机电流参数是否正确设置，若参数设置正确，再检查负载是否存在空载工作情况
E009 堵转保护 请检查电机电流参数是否正确设置，若参数设置正确，再检查负载是否存在堵转工作情况
E00A 电流不平衡 请检查输入电压是否不平衡，若输入电压平衡，再检查负载是否存在三相线间电阻不平衡
E00B 电机启动电流过大 请检查负载电机是否存在短路
E00C 计量芯片故障 电源保护器损坏，请更换设备
